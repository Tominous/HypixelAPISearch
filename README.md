# HypixelAPISearch
A simplistic Hypixel statistics search website created using <a href="https://github.com/Plancke/hypixel-php">Plancke's HypixelPHP</a> & designed with <a href="http://getbootstrap.com/">Bootstrap</a>.</br>

<b>Installation</b><br>
1. Upload files to webserver.<br>
2. Upload HypixelPHP (make sure to ammend location in 'hypixel.php' if changed.</br>
